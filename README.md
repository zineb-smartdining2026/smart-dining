# 🍜 Smart Dining

A web application for exploring dishes, adding favorites, and getting personalized recommendations.

## 🚀 Features

- Search dishes by name
- Filter dishes by category
- Add and remove favorites
- Rate dishes
- Get recommendations based on favorites
- View profile summary
- Admin panel to add and delete dishes

## 🛠️ Technologies Used

- Python
- Flask
- Flask-Login
- Flask-SQLAlchemy
- HTML
- CSS
- Bootstrap
- JavaScript
